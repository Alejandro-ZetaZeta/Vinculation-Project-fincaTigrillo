# Finca Tigrillo — Design Identity

## Colors

- **Emerald** `#16a34a`: Primary brand color, CTAs, icons, accent text
- **Emerald Light** `#22c55e`: Progress fills, highlight accents
- **Forest** `#0b2414`: Dark backgrounds, hero sections
- **Forest Mid** `#1a2414`: Secondary dark surfaces
- **Slate** `#374151`: Panel backgrounds for feature scenes
- **Parchment** `#f6f7f2`: Light/neutral backgrounds, community scene
- **White** `#ffffff`: Cards, light text on dark, brand reveal
- **Mint** `#d1fae5`: Subtle border accents on dark
- **Danger** `#fca5a5`: Pain point emphasis, alert text

## Fonts

- **Display:** `"Syne"` — headlines, brand name, large numbers. 400-900 variable.
- **Body:** `"DM Sans"` — body text, labels, metadata. 400-700 variable.
- **Fallback:** `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`

## Component Stylings

#### Primary CTA (Emerald)
- Background: `#16a34a`, Text: `#ffffff`, Font: DM Sans 16px/600
- Padding: `10px 16px`, Border radius: `12px`

#### KPI Card (dark variant)
- Background: `rgba(255,255,255,0.06)`, Border: `1px solid rgba(209,250,229,0.2)`
- Border radius: `24px`, Padding: `32px 20px`

#### Panel (slate/glass)
- Background: `rgba(0,0,0,0.25)`, Border: `1px solid rgba(255,255,255,0.12)`
- Border radius: `22px`, Padding: `28-32px`

#### Community Card (light)
- Background: `rgba(255,255,255,0.92)`, Border: `1px solid rgba(22,163,74,0.2)`
- Border radius: `22px`, Padding: `32px`

## Layout Principles

- Dark-first with emerald accents for feature scenes
- Light/neutral for community/education section
- Billboard-style: single hero element per beat on full-bleed background
- Generous whitespace: 50-80px edge padding, 20-28px between elements

## Do's & Don'ts

1. **DO** use Emerald (`#16a34a`) for all accent/highlight elements
2. **DO** keep dark scenes on Forest (`#0b2414`) backgrounds
3. **DO** use Syne for numbers and headlines only
4. **DON'T** put dark text on dark backgrounds — white text only on Forest/Slate
5. **DON'T** use more than 2 font sizes per beat — hierarchy through weight
