# Finca Tigrillo Mobile

Este directorio contiene una app móvil independiente basada en Expo.

## Objetivo

- Mantener el proyecto original intacto.
- Tener una app móvil separada que use la misma lógica de negocio y API en el futuro.

## Cómo usar

1. Abrir el directorio `mobile-app`.
2. Ejecutar `npm install`.
3. Iniciar con `npm run start`.
4. Escanear el código QR con Expo Go o usar el emulador.

## Nota

Esta es una base inicial. Las pantallas son placeholders que se pueden conectar a la API de `http://localhost:3000/api` o a tu backend existente.
